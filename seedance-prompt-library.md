## SeaDance 模型简介

SeaDance 是字节跳动（ByteDance）推出的 AI 视频生成模型系列，包含以下版本：

| 版本 | 特点 | 适用场景 |
|------|------|----------|
| SeaDance 1.0 Lite | 小参数模型，生成速度快，性价比高 | 快速迭代、社交短视频 |
| SeaDance 1.0 Pro | 大参数模型，支持多镜头叙事 | 电影感短片、广告 |
| SeaDance 1.5 Pro | 原生音视频联合生成，支持口型同步 | 对白场景、MV、剧情片 |
| SeaDance 2.0 | 最新版本，15秒视频，5个分镜 | 专业级内容创作 |

## 提示词公式

### 基础公式

```
提示词 = 主体 + 运动 + 场景 + 镜头/运镜 + 风格/美学
```

### Pro 版音频公式

```
提示词 = 主体 + 运动 + 场景 + 运镜/切镜 + 美学描述 + 声音描述
```

### 人物音色描述公式

```
音色 = 性别 + 年龄区间 + 声音属性 + 语速 + 情绪基线
```

## 镜头维度

### 1. 景别（Shot Size）

| 景别 | 英文 | 提示词 | 用途 |
|------|------|--------|------|
| **特写** | Close-up | 脸部特写、眼部特写、手部特写 | 表现情绪、强调细节 |
| **近景** | Medium Close-up | 胸像、半身 | 人物表情、姿态 |
| **中景** | Medium Shot | 腰部以上、膝部以上 | 人物动作、互动 |
| **全景** | Wide Shot | 全身、人物与环境 | 交代环境、展示动作 |
| **远景** | Extreme Wide | 大远景、航拍 | 营造氛围、展示规模 |

### 2. 运镜方式

| 运镜 | 提示词 | 效果描述 |
|------|--------|----------|
| **推镜** | 镜头推进、缓慢推近、急速前推 | 聚焦主体，增强紧张感 |
| **拉镜** | 镜头后拉、缓慢拉远 | 展现场景，产生疏离感 |
| **摇镜** | 镜头左摇、右摇、快速摇摄 | 跟随动作，展现场景 |
| **移镜** | 横移、平移、轨道移动 | 跟随主体，保持构图 |
| **跟拍** | 跟随拍摄、跟踪镜头 | 保持主体在画面中 |
| **环绕** | 环绕拍摄、360度环绕、缓慢旋转 | 展示主体全貌 |
| **升降** | 镜头上升、下降、升降镜头 | 改变视角，营造气势 |
| **手持** | 手持镜头、轻微抖动、呼吸感 | 增加真实感、纪录片风格 |
| **航拍** | 俯拍、鸟瞰、无人机视角 | 展示大环境 |
| **变焦** | 快速变焦、缓慢变焦、希区柯克变焦 | 视觉冲击、心理暗示 |

### 3. 镜头切换

```
提示词 = 镜头1描述 + "镜头切换" + 镜头2描述
```

**示例：**
- "全景拍摄，模特微笑着走近镜头。镜头切换，近景拍摄模特下半身"
- "特写老人眼睛。镜头切换，中景老人整体姿态。镜头切换，全景老人在咖啡厅"

**切镜时机控制：**
- "在动作进行到一半时切镜"
- "当人物转头时切换镜头"
- "对话结束后切到反应镜头"

| 参数 | 选项 | 提示词示例 |

## 声音与音效（Pro 版）

SeaDance 1.5 Pro 支持原生音视频联合生成，可以在提示词中描述声音。

### 1. 人声对白

```
描述格式："[角色]说：[对白内容]，[语言]，[情绪]，[语速]"
```

**示例：**
- "年轻女性说：'我等你很久了'，普通话，温柔，语速缓慢"
- "中年男性说：'我们走吧'，英文，低沉，语速正常"
- "老人说：'那时候我们还年轻'，四川话，怀旧，语速缓慢"

### 2. 支持的语言与方言

| 类别 | 支持 |
|------|------|
| **中文** | 普通话、陕西话、四川话、粤语等 |
| **外语** | 英语、日语、韩语、西班牙语、印尼语等 |

### 3. 环境音效

| 类型 | 提示词 |
|------|--------|
| **自然** | 鸟鸣、风声、雨声、海浪声、雷声 |
| **城市** | 车流声、地铁声、人群嘈杂、施工声 |
| **室内** | 键盘敲击、翻书声、咖啡机运作、空调声 |
| **动作** | 脚步声、敲门声、开门声、碰撞声 |

### 4. 背景音乐

```
提示词 = "背景音乐：[风格]，[情绪]，[乐器]"
```

**示例：**
- "背景音乐：钢琴独奏，忧伤，慢节奏"
- "背景音乐：电子音乐，未来感，鼓点强劲"
- "背景音乐：管弦乐，史诗感，气势磅礴"

## 进阶技巧

### 1. 人物一致性技巧

- 使用"三视图"参考图确保人物外貌一致
- 在多个提示词中重复关键外貌描述词
- 使用"@"功能引用参考图锁定角色

### 2. 避免肢体崩坏

- 避免过度强调手部和脚部特写
- 使用"自然姿态"、"放松状态"等描述
- 多次生成抽卡选择最佳效果

### 3. 多镜头连贯性

```
全局风格（适用于所有镜头）：
[主体外貌] + [服装] + [场景] + [色调] + [光线]

切镜约束：
- 人物始终位于画面中央
- 服装保持不变
- 光线方向保持一致
```

### 4. 负面提示词说明

⚠️ **注意：** SeaDance 模型**不响应负面提示词**，不要浪费时间写 "avoid..."、"no..."、"without..." 等描述。

正确做法是在正面描述中明确你想要什么，而不是不想要什么。

### 5. 图生视频优化

- 描述运动而非静止状态
- 明确指出运动的主体
- 避免与图片内容矛盾的描述
- 使用程度副词强调运动强度
