# 项目二：AI 视频生成探索

## 项目定位
探索 AI 视频生成技术的实践应用，测试不同平台（千问、豆包、可灵等）的效果，积累提示词经验和 workflow。

## 目标
- 掌握主流 AI 视频生成工具的使用方法
- 建立有效的提示词体系（Prompt Engineering）
- 形成可复用的视频生成 workflow
- 记录不同平台的特性与适用场景

## 文件结构
```
project-ai-video/
├── README.md          # 本文件
├── prompts/           # 视频生成提示词库
├── workflows/         # 生成流程与参数记录
├── references/        # 参考案例与学习资料
└── outputs/           # 生成的视频素材、截图
```

## 相关平台
- 通义万相（阿里千问）

- 快手可灵
