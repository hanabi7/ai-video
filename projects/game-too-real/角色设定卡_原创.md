# 《末日游戏公司》角色设定卡（原创）

## 主要角色

### 🎭 林默（主角）

| 项目 | 内容 |
|-----|------|
| **身份** | 地下城拾荒者/避难所管理员/"游戏策划" |
| **年龄** | 23岁 |
| **外貌** | 清瘦结实，眼神冷静内敛，黑色短发 |
| **服装** | 改装防护服+旧风衣，实用主义风格 |
| **标志性道具** | 全息平板（系统终端，左手腕佩戴） |
| **性格标签** | 闷骚、理智、内心吐槽狂、擅长忽悠 |
| **经典表情** | 表面平静，内心弹幕刷屏 |
| **声优建议** | 低沉中带点无奈的青年音 |

**设计要点**：
- 外表看起来可靠但有点疲惫（管理避难所很累）
- 眼神要有"我看透一切"的冷静感
- 系统终端是发光的全息平板界面
- 站姿沉稳，有领袖气质但不太正经

## 变异生物设定

### 辐射犬（初级怪物）

| 项目 | 内容 |
|-----|------|
| **原型** | 流浪狗受辐射变异 |
| **体型** | 比狼还大一圈 |
| **特征** | 皮肤溃烂结痂、发光蓝眼、獠牙外露 |
| **特殊** | 身上有辐射荧光纹路 |
| **行为** | 群体行动，极其凶残 |

**设计要点**：
- 恶心但不至于恐怖（毕竟是"游戏"）
- 蓝色发光眼睛是标志
- 身上有变异增生组织
- 动作野兽化，但保留犬类特征

## 场景设定

### 避难所（主要场景）

**整体风格**：地下工业风+科幻全息

**主要区域**：
1. **控制室**：林默工作地点，满墙屏幕，蓝色氛围灯
2. **传送室**：三个休眠舱，光束从天而降
3. **大厅**：玩家聚集地，工业风家具，任务公告板
4. **宿舍区**：简易床位，有生活气息

**色调**：
- 主色调：深蓝、银灰（科技感）
- 点缀色：橙色（警告灯）、青色（全息光）
- 与外面废土形成冷暖对比

---

### 废土（外部场景）

**整体风格**：末日废土+赛博残骸

**关键元素**：
- 废弃高楼，钢筋外露
- 变异植物（发光藤蔓）
- 破旧车辆，风滚草
- 昏黄天空，辐射云
- 远处废墟有变异兽身影

**色调**：
- 主色调：暗黄、灰绿（死寂）
- 点缀色：霓虹残光（战前遗留）
- 天空：昏黄+绿色辐射云

---

## 美术风格

### 整体风格
- **主体**：写实渲染+动漫角色
- **UI元素**：赛博朋克全息风
- **色调对比**：
  - 避难所内部：冷色调（蓝、银）
  - 废土外部：暖黄+灰绿（荒芜）

### 参考作品
- 《辐射》系列 - 废土美学
- 《赛博朋克2077》 - 霓虹科技
- 《灵笼》 - 国漫废土
- 《边缘行者》 - 赛博朋克风格

### 特效需求
1. **系统界面**：全息投影，半透明，蓝紫色发光
2. **传送光束**：能量光柱，粒子效果
3. **变异兽眼睛**：蓝色发光
4. **辐射尘埃**：空气中漂浮微粒
