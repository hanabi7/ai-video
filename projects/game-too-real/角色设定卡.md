# 《这游戏也太真实了》角色设定卡

## 主要角色

### 🎭 楚光（主角）

| 项目 | 内容 |
|-----|------|
| **身份** | 避难所管理员 / "游戏策划" |
| **年龄** | 22岁 |
| **外貌** | 瘦削精悍，眼神锐利，黑色短发 |
| **服装** | 破旧但整洁的避难所制服（深蓝色连体工装） |
| **标志性道具** | 发光的系统终端（左手腕佩戴） |
| **性格标签** | 冷静、理智、吐槽狂、擅长忽悠 |
| **经典表情** | 表面微笑，内心MMP |
| **声优建议** | 沉稳中带点无奈感的青年音 |

**设计要点**：
- 外表看起来可靠但有点疲惫（毕竟管理避难所很累）
- 眼神要有"我看透一切"的感觉
- 系统终端要有科技感，发光的界面
- 站姿要有领袖气质但不要太正经（毕竟是"游戏策划"人设）

## 变异生物设定

### 变异犬（初级怪物）

| 项目 | 内容 |
|-----|------|
| **原型** | 流浪狗变异 |
| **体型** | 比普通狗大1.5倍 |
| **特征** | 皮肤溃烂、发光红眼、獠牙外露 |
| **特殊** | 身上有辐射斑纹，发出低吼 |
| **行为** | 群体行动，攻击性极强 |

**设计要点**：
- 要恶心但不能太恐怖（毕竟是"游戏"）
- 红色发光眼睛是标志性特征
- 身上要有明显的变异痕迹
- 动作要野兽化，但保留狗的基本形态

## 场景设定卡

### 避难所（主要场景）

**整体风格**：工业风+科幻感

**主要区域**：
1. **控制室**：楚光的工作地点，满是屏幕和按钮，蓝色灯光
2. **传送室**：三个休眠舱，传送光束从天而降的地方
3. **大厅**：玩家们聚集的地方，工业风家具，白板（贴满任务）
4. **宿舍区**：简陋的床铺，但有生活气息

**色调**：
- 主色调：深蓝、灰色（科技感）
- 点缀色：橙色（警告灯）、白色（灯光）
- 与外面的废土形成温暖vs寒冷的对比

---

### 废土（外部场景）

**整体风格**：末日废土+赛博朋克

**关键元素**：
- 废弃的高楼大厦，钢筋外露
- 到处都是变异植物（发光的藤蔓）
- 破旧的车辆，风滚草
- 灰暗的天空，偶尔有酸雨
- 远处的废墟中有变异兽的影子

**色调**：
- 主色调：灰黄、暗绿（死寂感）
- 点缀色：霓虹灯的残光（战前遗留）
- 天空：昏黄+辐射云

## 美术风格参考

### 整体风格
- **主体**：3D写实渲染
- **UI元素**：赛博朋克全息风
- **色调对比**：
  - 避难所内部：暖色调（希望、安全）
  - 废土外部：冷色调（危险、荒芜）

### 参考作品
- 《辐射4》（Fallout 4）- 废土美学
- 《赛博朋克2077》 - 霓虹与科技
- 《头号玩家》 - 游戏与现实的边界
- 《灵笼》 - 国漫废土风格

### 特殊效果需求
1. **系统界面**：全息投影效果，半透明，蓝色发光边框
2. **传送光束**：光柱效果，粒子散落
3. **变异兽眼睛**：红色发光，有点恐怖但不至于噩梦
4. **辐射尘埃**：空气中漂浮的微粒，昏黄的滤镜
