# SeaDance Prompt Library

A comprehensive prompt library for ByteDance's SeaDance AI video generation model.

## Overview

This library provides a systematic collection of prompts organized by dimensions:
- **Character** - Appearance, clothing, emotions, actions
- **Camera** - Shot sizes, movements, angles, transitions
- **Style** - Cinematic looks, art styles, lighting, color grading
- **Scene** - Indoor, outdoor, weather, time of day
- **Audio** - Voice, sound effects, background music (Pro version)

## Quick Start

### Basic Prompt Formula
```
Prompt = Subject + Motion + Scene + Camera/Style
```

### Pro Audio Formula
```
Prompt = Subject + Motion + Scene + Camera + Aesthetics + Sound
```
